Project by [Tina Milosavljevic](https://github.com/tinam02/moshi-monsters) adapted for npm.

#### Usage

```bash
bun add iggy-cursor
ln -s ../node_modules/iggy-cursor/assets/ public/iggy
```

```typescript
import { iggyCursor } from 'iggy-cursor';

iggyCursor();
```
